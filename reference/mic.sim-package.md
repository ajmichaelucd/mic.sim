# mic.sim: Simulating and Fitting Models to MIC Data

mic.sim includes functions for analyzing minimum inhibitory
concentration data, including functions for importing or simulating
data, viewing the distribution, model fitting, and model visualization.
The model fitting uses a latent Gaussian mixture model framework to
analyze trends in the mixture weights and component means on the log2
MIC scale to attempt to distinguish and model the wild type and non-wild
type subpopulations.
